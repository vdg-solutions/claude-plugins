# vdg-solutions Claude Code plugin marketplace

Add this marketplace and install plugins:

```bash
claude plugin marketplace add vdg-solutions/claude-plugins
claude plugin install memctl@vdg-solutions
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [memctl](https://github.com/vdg-solutions/memctl-releases) | Zero-config persistent memory for Claude Code — auto-capture, auto-inject, vault-backed |
